사다리, 돌림판, 번호 뽑기를 한곳에서.

팀 순서 정하기, 이벤트 당첨자 선정, 게임 벌칙 정하기, 발표 순서 뽑기처럼 빠른 랜덤 선택이 필요할 때 바로 쓰는 브라우저 추첨 도구입니다.

What this site does

가볍게 쓰는 공정한 랜덤 선택 도구

랜덤툴 스튜디오는 복잡한 가입이나 설치 없이 사다리타기, 돌림판, 번호 뽑기를 바로 실행할 수 있게 만든 웹 도구입니다. 입력한 이름과 항목은 브라우저 안에서 처리되며, 회의, 방송, 수업, 모임, 이벤트처럼 결과를 빠르게 보여줘야 하는 상황에 맞춰 만들었습니다.

직접 입력

참가자 이름, 결과, 돌림판 항목, 번호 범위를 직접 바꿔서 상황에 맞는 추첨을 만들 수 있습니다.

보이는 진행

사다리는 직접 선택한 줄이 내려가는 과정을 보여주고, 돌림판은 회전 후 포인터가 가리키는 결과를 확인할 수 있습니다.

여러 언어 지원

한국어, 영어, 일본어, 스페인어 화면을 제공해서 다양한 사용자와 함께 사용할 수 있습니다.

Ladder game

사다리 타기

참가자와 결과를 같은 화면에서 입력하고, 위쪽 번호를 눌러 한 명씩 사다리를 탈 수 있습니다. 전체 결과는 별도 버튼으로 확인합니다.

위쪽 참가자 번호를 누르면 사다리를 타고 내려갑니다.

Spin wheel

돌림판 추첨

같은 항목을 여러 칸으로 복제하지 않고, `커피 x2`처럼 가중치를 입력하면 하나의 큰 조각으로 표시됩니다.

x2, *3는 조각을 여러 개 만들지 않고 그만큼 큰 조각으로 만듭니다.

Number draw

범위 번호 뽑기

1번부터 30번처럼 원하는 시작 번호와 끝 번호를 정하고, 제외할 번호를 빼고 필요한 개수만 뽑을 수 있습니다.

결과 아직 추첨 전입니다.

Guide

사용 전에 알아두면 좋은 점

어떤 상황에 쓰나요?

방송 벌칙, 발표 순서, 회식 메뉴, 이벤트 추첨, 게임 순서, 수업 활동처럼 가벼운 무작위 선택에 맞습니다.

입력한 이름이 저장되나요?

도구 입력값은 추첨을 진행하기 위해 브라우저에서 사용됩니다. 별도 서버 데이터베이스에 저장하도록 만든 사이트가 아닙니다.

가중치 입력은 어떻게 하나요?

돌림판 항목 뒤에 `x2` 또는 `*3`처럼 적으면 해당 항목이 더 큰 조각으로 표시됩니다.

공식 추첨에 써도 되나요?

가벼운 선택 도구로 제공됩니다. 법적, 금융, 계약, 공식 경품 추첨처럼 검증 절차가 필요한 용도에는 별도 기준을 확인하세요.
